# Relógio Mundial App

Aplicação web desenvolvida em React que exibe a hora local em dois formatos (digital e analógico), com interface dinâmica e temas visuais que mudam ao longo do dia.

## Visão Geral

O projeto apresenta um relógio interativo com foco em experiencia visual:

- Modo digital com tipografia numérica otimizada para leitura.
- Modo analógico com ponteiros de hora, minuto e segundo.
- Alternância entre os modos por botão.
- Cenário dinâmico por horário (manha, tarde, entardecer e noite).
- Elementos animados: sol, lua, nuvens, estrelas e efeito de brilho noturno.
- Exibição do fuso horário detectado automaticamente pelo navegador.
- Exibição da data atual em formato PT-BR.

## Tecnologias Utilizadas

- React 19
- JavaScript (ES6+)
- CSS3 (animações, gradientes e responsividade)

## Como Executar o Projeto

### 1. Pré-requisitos

- Node.js 18+ (recomendado)
- npm 9+

### 2. Instalar dependências

```bash
npm install
```

### 3. Rodar em ambiente de desenvolvimento

```bash
npm start
```

A aplicação ficara disponível em:

http://localhost:3000


## Possíveis Evoluções

- Adicionar seleção manual de fusos horários para simular um relógio realmente mundial.
- Persistir preferencia do modo (digital/analógico) no navegador.
